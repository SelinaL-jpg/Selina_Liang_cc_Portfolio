Autobiographical Game

For this project, I designed Cats' Secret Mission, a game with three levels that reflect my personal story. To manage the game, I used an array to store the levels as objects, with a currentLevel variable controlling transitions. This approach kept the code modular and easy to update. I also implemented the player as a class, which allowed me to centralize its behavior and properties. Using the switch statement, I ensured smooth transitions between scenes, and simple keyboard controls made the game intuitive for players. Challenges and Learning Organizing interactions across levels was a challenge, but using classes and arrays simplified the process. Debugging also helped me better understand user experience design. I learned to: Manage complex game logic using classes and objects. Create modular, scalable code for a multi-level game. Balance technical functionality with creative storytelling. Future Improvements In the future, I plan to add sound effects and animations to make the game more engaging and visually polished. This project allowed me to explore personal storytelling through game design while improving my technical and creative skills.
