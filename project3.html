Experimental_Clock

In this project, I used p5.js to create a dynamic visual effect based on time. The rectangles in different sections change color and opacity over time using the second(), minute(), and hour() functions. Each time unit (seconds, minutes, hours) is mapped to a range of hue and opacity using the map() function, resulting in smooth gradient transitions.
