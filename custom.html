Learning Notes

Exploring P5.js

Spring 2024

Learning P5.js opened a door for me to experiment with code as a creative tool. Starting with basic shapes and gradually building complexity, I discovered how programming can merge with art. Projects like Optical Illusion and Face Generator allowed me to visualize abstract ideas in an interactive form. Through these exercises, I’ve gained confidence in using code to explore generative art and dynamic interactivity, pushing beyond static design.

This process has also deepened my understanding of animation and interactivity—valuable concepts that I can integrate into future UI/UX projects and immersive experiences.

Creative Coding Insights

Learning Summary

Throughout this semester, I learned that creative coding is about balancing precision with play. Key insights I gained include:

  • Start Simple, Iterate Often: Beginning with minimal shapes or basic logic and layering on complexity step by step.
  • Experimentation as a Tool: Adjusting parameters and adding randomness often led to unexpected but exciting results.
  • Clean Code, Clear Ideas: Writing organized, readable code not only improves efficiency but also makes creative exploration smoother.

By blending my knowledge of design principles with coding experimentation, I discovered new ways to tell stories and create immersive visual experiences.
